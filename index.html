Projects

  • RecyclEd - Python

    RecyclEd is a reverse recycling machine that takes recyclable items from users and classifies them according to their object category (plastic bottle, glass bottle, metal can, or rubbish). Object classification is performed using a residual neural network. If the object is classified as recycling, the system will move the object into the appropriate bin. Based on the items accepted, the user will be entitled to monetary compensation, given through a coupon to the user.

    Find out more in the blog post!

  • Drone Food Delivery Service - Java

    In this day and age, many people, including students, have their food delivered. Wouldn't it be nifty if we could create a drone that would deliver meals to precise coordinate locations?

    This task of this project was to devise an algorithm that would act as cost-constrained path finder: it would plan the route that the drone would take to deliver the food orders on a given day, maximing the daily profit.

    Find out more in the blog post!

  • The Effect of COVID-19 on the Usage of 'Just Eats Cycles' - Python

    In a report with a coursework partner, we investigate how the usage of the bike sharing service ’Just Eat Cycles’ was impacted by the COVID-19 pandemic. We hypothesised that users of ’Just Eat Cycles’ would use this service to visit scenic locations and to exercise, instead of commuting.

    See the full report on GitHub.

  • Arduino Audio Visualiser - C/C++

    Using an ESP32 board, I connected two microphones and an OLED dislplay to the microcontroller. With these, I processed the input from the analogue microphone and displayed the signal on the display according to the volume intensity.
